EN: 

By default you can only access CFW in emmunand to prevent bans.

ES:

Por defecto solo le puede iniciar el CFW en emunand para prevenir bans.

Para activar el CFW en sysnand tambien, ejecuta cfw_sysnand.bat

Para volver a solo emunand, simplemente ejecuta cfw_emunand.bat
